# Task 3

下载 软科中国大学专业排名 bcsr

在浏览器开发者工具里网络中查找 http://127.0.0.1:8651，后面跟的 url 就是 API 接口。

中国大学专业排名 API 接口

https://www.shanghairanking.cn/api/pub/v1/bcsr/rank?target_yr=2022&subj_code=0101

https://www.shanghairanking.cn/api/pub/v1/bcsr/subj?year=2022
